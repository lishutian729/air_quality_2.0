关于本项目

项目简介

本项目是一个基于机器学习的空气质量预测系统，旨在为用户提供准确的空气质量预测信息， 帮助人们更好地了解和应对空气污染问题。

主要功能

  • 实时空气质量监测与预测
  • 历史数据查询与分析
  • 数据可视化展示
  • 多模型集成预测

技术栈

后端技术
  • Python
  • Flask
  • Scikit-learn
  • LightGBM
前端技术
  • Bootstrap 5
  • JavaScript
  • Chart.js

数据来源

本系统使用的数据来自多个来源：

  • 环境空气质量自动监测数据
  • 气象数据
  • 历史统计数据

预测模型

系统采用多模型集成的方式进行预测，主要包括：

随机森林

适用于处理非线性关系，具有良好的泛化能力。

LightGBM

高效的梯度提升框架，具有出色的预测性能。
